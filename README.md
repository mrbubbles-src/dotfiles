# mrbubbles-src dotfiles

## Usage

### MacOS

- Install `Homebrew`:
	- `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
- Install `chezmoi`
	- `brew install chezmoi`
- Apply ´dotfiles´
	- `chezmoi init --apply mrbubbles-src`
